# cinema-web-research-archive
本项目用于收集和整理影视类网站的公开研究型基础信息，对电影、电视剧及综合影视站点进行统一归档。仓库内容仅限站点信息层面的记录，不包含任何影视资源或播放入口。所有整理行为基于公开资料完成，项目定位为中性的信息整理与研究工具，适合长期查询、对照与资料留存。
